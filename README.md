# Smart Kids Podcast
🎙️ Plataforma educativa de podcasts para estudiantes de primaria

## Demo
- **Sitio:** https://smart-kids-podcast.vercel.app
- **Admin:** https://smart-kids-podcast.vercel.app/#/admin

## Tecnologías
React 19 • TypeScript • Firebase • Vercel • Tailwind CSS

## Características
- ✅ Panel administrativo completo
- ✅ CRUD de episodios, profesores y testimonios
- ✅ Integración con Spotify
- ✅ Dashboard con estadísticas
- ✅ Autenticación segura
- ✅ Diseño responsive

## Instalación
```bash
npm install
npm run dev
```

## Deploy
```bash
vercel --prod
```
